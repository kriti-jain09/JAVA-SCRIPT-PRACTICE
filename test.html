JS Display Possibilities

JS can display data in 4 different ways

  • writing in HTML element, using innerHTML.
  • writing into the HTML output, using document.write().
  • writing into an alert box, using window.alert().
  • writing into the browser console, using console.log().

Using innerHTML

to access an HTML element JS can use document.getElementById(id)
the id attribute defines the element and innerHTML property defines the content

Using document.write()

if document.write() is used after documentis loaded, it will delete all existing HTML
this method should only be used for testing

Using window.alert()

we can use alert box to display data

we can skip the window keyword
In JavaScript, the window object is the global scope object, that means that variables, properties, and methods by default belong to the window object.

Using console.log()

for debugging purposes this can be used in browser to display data

JS Print

JS doesn't have any print methods.
we cannot access output devices using JS.
